# BandSheet 新手上手文件

歡迎加入 BandSheet 開發!這個資料夾是專門寫給 **React / Java 新手** 的「保姆級」教學。
就算你只寫過一點點程式,只要照著讀、照著做,也能理解整個專案、跑起來、並開始貢獻。

## 這套文件在講什麼

BandSheet 是一個「讓樂團即時共編和弦歌詞譜」的網站。它同時用到:
- **後端**:Java + Spring Boot(處理資料、登入、API)
- **前端**:React + TypeScript(畫面、互動)
- **資料庫**:PostgreSQL(存資料)
- 全部用 **Docker** 一鍵跑起來

你會透過這個真實專案,學到「一個現代網站從瀏覽器到資料庫是怎麼運作的」。

## 一句話心法

> 不用一次看懂全部。先讓專案「跑起來」,再挑一個你看得到的畫面(例如「我的歌曲」),
> 從畫面一路往後追到資料庫,你就會懂了。這套文件就是帶你這樣追一遍。
